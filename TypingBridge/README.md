# **<typingbridge.pyについて><br>**
typingbridgeはPCに接続されたキーボードの入力信号をM5Core2へ中継するハブとなるアプリケーションです。<br>USBシリアル通信/Bluetooth、RawHIDを用いての通信を成立させます。
